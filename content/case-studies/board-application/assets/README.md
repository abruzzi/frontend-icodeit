# Assets (`board-application`)

Diagrams, screenshots, or other static files for this case study.

**Serving in the app:** put files you need at a URL under `public/` (for example `public/case-studies/board-application/diagram.png` → `/case-studies/board-application/diagram.png`), or import from `public` in a component under `components/case-studies/board-application/`.

Keep source-of-truth art here when you want it versioned next to the MDX.
